#  Yuumi - Site de Receitas


Yuumi é um protótipo de um site de receitas, desenvolvido como projeto final para a disciplina de **Desenvolvimento Web 2**. O projeto foi desenhado no Figma e implementado utilizando HTML, Tailwind CSS e JavaScript, com foco na criação de uma interface limpa, moderna e responsiva.

## Material Design
O design do Yuumi foi pensado para criar uma experiência de usuário limpa e profissional, com a aparência de um aplicativo real. Para isso, foi definida uma identidade visual coesa, inspirada em princípios de design modernos.

* **Paleta de Cores:** Foi usada uma paleta consistente com o amarelo (brand-yellow) como cor de destaque e um preto suave (brand-light-black) para textos, garantindo uma identidade visual marcante e legível.
* **Hierarquia Clara:** As páginas são organizadas de forma lógica, com seções bem definidas (como a busca principal, a receita em destaque e a galeria) para guiar o usuário de forma intuitiva.
* **Design Responsivo:** O layout se adapta fluidamente a qualquer tamanho de tela, do celular ao desktop, garantindo uma boa experiência em todos os dispositivos.
  
## Funcionalidades Principais

* **Página Inicial Dinâmica:** Apresenta uma seção de busca com imagem de fundo, uma "Receita do Dia" em destaque e uma galeria de "Receitas Populares".
* **Design Responsivo:** O layout se adapta a diferentes tamanhos de tela, de celulares a desktops, utilizando a metodologia "mobile-first".
* **Componentização:** O Header e o Footer são tratados como componentes separados e injetados dinamicamente nas páginas com JavaScript, evitando a repetição de código.
* **Páginas Essenciais:** Inclui protótipos para a página inicial, uma página de resultados de busca, uma página de detalhes da receitas e uma página para filtro de receitas.

## Tecnologias Utilizadas

* **HTML5** para a estruturação do conteúdo.
* **Tailwind CSS**  para a estilização de componentes e responsividade.
* **JavaScript** para a funcionalidade de carregamento dinâmico dos componentes.
* **Figma** para o design e prototipação da interface.
* **SimpleIcons** para os icones de marca.

## Como Executar o Projeto

Para visualizar este projeto localmente, é **essencial** usar um servidor web local. Abrir o arquivo `index.html` diretamente no navegador não funcionará corretamente.

**A maneira mais fácil de executar é com a extensão "Live Server" no Visual Studio Code:**

1.  **Clone o repositório:**
    ```bash
    git clone [https://github.com/nathaliabg12/DSW2-Yuumi.git](https://github.com/nathaliabg12/DSW2-Yuumi.git)
    ```
2.  **Abra a pasta** do projeto no Visual Studio Code.
3.  **Instale a extensão `Live Server`** (se ainda não tiver).
4.  Clique com o **botão direito** no arquivo `index.html` e selecione **"Open with Live Server"**.

O site será aberto no seu navegador com um endereço como `http://127.0.0.1:5500`.
